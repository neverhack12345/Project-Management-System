# MCP Examples

Numbered MCP walkthroughs for assistants (daily triage, playbooks, CRUD, vault, facts) are maintained in the **Agent Skills** reference bundles so they load on demand with each skill:

- [.cursor/skills/markdown-projects/references/mcp-examples.md](../.cursor/skills/markdown-projects/references/mcp-examples.md)
- [.cursor/skills/second-brain/references/mcp-examples.md](../.cursor/skills/second-brain/references/mcp-examples.md)

The two files are the same snapshot. In Cursor, prefer activating **markdown-projects** or **second-brain** and opening `references/mcp-examples.md` from `references/REFERENCE.md` when you need step-by-step call sequences.

For tool names and parameters, see [MCP_TOOL_REFERENCE.md](./MCP_TOOL_REFERENCE.md). For client wiring and safety rules, see [MCP_USAGE.md](./MCP_USAGE.md).
